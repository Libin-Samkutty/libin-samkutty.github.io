api-test-data-generator  

Python Package PyPI Published Open Source

Stack: Python, Faker, pytest, JSON/YAML

Every API test suite I worked on had the same problem: static test data fixtures that needed manual updates whenever the schema changed. The package came out of that; define your schema once in JSON or YAML, generate realistic records on demand, and plug directly into pytest fixtures.

Key Capabilities:
• Schema-defined generation from JSON/YAML: field types include strings, integers, emails, UUIDs, dates, phone numbers, and nested objects.
• Faker integration for realistic names, addresses, and company data; seed support for reproducible datasets.
• Export to JSON or CSV; native pytest fixture compatibility for immediate test suite integration.
• CLI and programmatic APIs for flexible usage in any pipeline.

Install:
pip install api-test-data-generator
